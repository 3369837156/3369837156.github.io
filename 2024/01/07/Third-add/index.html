解题思路：
这道题可以使用滑动窗口的方法来解决。我们可以使用一个哈希表来记录每个字符最后一次出现的位置。同时，我们使用两个指针left和right来表示滑动窗口的左右边界。我们遍历字符串s，每次将right指针向右移动一位，并将当前字符加入哈希表中。如果当前字符已经在哈希表中出现过，并且其最后一次出现的位置在left和right之间，我们将left指针移动到该位置的下一位。然后，我们更新最长子串的长度。最后，我们返回最长子串的长度。

var lengthOfLongestSubstring = function(s) {
let map = new Map();
let left = 0;
let max = 0;
for(let i = 0; i < s.length; i++) {
if(map.has(s[i]) && map.get(s[i]) >= left) {
left = map.get(s[i]) + 1;
}
map.set(s[i], i);
max = Math.max(max, i - left + 1);
}
return max;
};
代码解释：
首先，我们创建一个空的哈希表map来记录每个字符最后一次出现的位置。然后，我们初始化左指针left为0，最大长度max为0。接下来，我们遍历字符串s的每个字符。对于每个字符，我们首先判断它是否已经在哈希表中出现过，并且其最后一次出现的位置在left和当前位置之间。如果是，我们将left指针移动到该位置的下一位。然后，我们将当前字符加入哈希表中，并更新其最后一次出现的位置为当前位置。最后，我们更新最长子串的长度，即max = Math.max(max, i - left + 1)。最后，我们返回最长子串的长度作为结果。
