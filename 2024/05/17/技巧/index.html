  1. 只出现一次的数字

给你一个 非空 整数数组 nums ，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

你必须设计并实现线性时间复杂度的算法来解决此问题，且该算法只使用常量额外空间。

示例 1 ：

输入：nums = [2,2,1]
输出：1
示例 2 ：

输入：nums = [4,1,2,1,2]
输出：4
示例 3 ：

输入：nums = [1]
输出：1

提示：

1 <= nums.length <= 3 * 104
-3 * 104 <= nums[i] <= 3 * 104
除了某个元素只出现一次以外，其余每个元素均出现两次。

class Solution {
public int singleNumber(int[] nums) {
// 初始化一个变量 single 为 0，用于存储最终只出现一次的数字
int single = 0;
// 遍历数组 nums 中的每个元素
for (int num : nums) {
// 将当前元素 num 与 single 进行异或运算，结果存回 single
// 由于异或运算的性质，重复的数字会相互抵消，最后 single 中存储的就是只出现一次的数字
single ^= num;
}
// 返回最终只出现一次的数字
return single;
}
}
